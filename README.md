# Proekt1
на МДК 2.2 Инструментальные средства разработки ПО мы создавали приложение, так же создавали дизайн самого приложения.  
Данный код реализует логику авторизации пользователя через JavaFX: проверяет логин и пароль, открывает главное окно при успехе или предлагает регистрацию при ошибке. Также контроллер следит за корректной инициализацией интерфейса.
В классе объявлены поля, связанные с элементами интерфейса: два текстовых поля (login, password) и кнопка (enter).
Также есть объект DBConnector, который отвечает за взаимодействие с базой данных.
 Так же есть метод OnActionEnter который  вызывается при нажатии на кнопку «Войти»:
Есть проверка данных: введённые логин и пароль передаются в метод TableConn класса DBConnector. Если метод возвращает true (данные верны), выполняется переход к главному окну,переход к главному окну:
Текущее окно закрывается, загружается новый FXML-файл (main.fxml), создаётся новая сцена и отображается главное окно.
Если данные неверны: программа выдает ошибку 
Пользователю предлагается попробовать зарегистрироваться.
Если пользователь выбирает «Зарегистрироваться», открывается окно регистрации (registration.fxml).
Также в этом классе есть метод initialize
Этот метод вызывается при инициализации контроллера: котрый проверяет, что все элементы интерфейса корректно связаны с FXML-файлом (через fx:id).
Если какой-то элемент не найден, появится ошибка с указанием, что не так с FXML.
